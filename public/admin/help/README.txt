Admin Help — optional screenshots
===================================

Drop PNG or WebP files here to replace the illustrated mockups on the Help page.
Use these exact filenames (lowercase):

  dashboard.png          — Main dashboard / analytics
  inventory-catalog.png  — Product catalog tab
  inventory-bulk.png     — Bulk shelving tab
  sales-orders.png       — Orders / picker queue
  sales-returns.png      — Returns management
  pos-sale.png           — POS sale mode
  pos-return.png         — POS return (48h) mode
  marketing-broadcast.png — Broadcasts
  marketing-analytics.png — Delivery analytics
  customers.png          — Customer list
  settings.png           — Settings
  super-root.png         — Super “Root Control” area (optional)

Images should be ~1200–1600px wide for clarity. The Help page shows your image when the file exists; otherwise it shows a built-in window illustration.
